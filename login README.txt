Description:

This project provides a basic login page designed using HTML and CSS. It includes simple validation for the username and password fields. If the username is set to 'santhosh' and the password is set to 'santhosh00777', the page will redirect to login2.html.

Usage:

Clone or download the project to your local machine.

Open the index.html file in a web browser.
